# mo_protobuf

Pure-Mojo runtime for the protobuf wire format.

## Contents

- `ProtoReader` - length-delimited, varint, fixed32/64, zigzag, skip_field
- `ProtoWriter` - all 15 scalar wire methods, nested message framing
- `ProtoSerializable` trait - `parse(mut reader) raises -> Self` and `serialize(self, mut writer) raises`
- `Bytes`, `WireType`, `FieldNumber`, `VarInt` aliases from `common.mojo`

Zero external dependencies. Used as the target runtime for code generated by [`protoc-gen-mojo`](https://github.com/kivicode/mo_grpc/tree/main/packages/protoc-gen-mojo).

## Install

```bash
pixi add mo_protobuf
```

## Example

```mojo
from mo_protobuf import ProtoReader, ProtoWriter, ProtoSerializable

var w = ProtoWriter()
w.write_string(1, "hello")
w.write_int32(2, 42)
var bytes = w.flush()

var r = ProtoReader(bytes^)
while r.has_more():
    var fnum, wtype = r.read_tag()
    if fnum == 1:
        print(r.read_string())
    elif fnum == 2:
        print(r.read_int32())
```
